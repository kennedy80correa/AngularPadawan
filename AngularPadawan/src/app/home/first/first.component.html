Descrição do curso


No curso de Introdução a lógica de Programção, tem como objetivo apresentar aos alunos noções básicas de algoritmo, sendo esse o pré-requisito para aprender as liguagens de programação.

Um algoritmo é uma estruturação passo-a-passo de como um determinado problema deve ser resolvido de forma não-ambígua (ou como muitos comparam "uma receita de bolo").

Com a conclusão desse curso os alunos devem ter a capacidade de desenvolver soluções passo-a-passo para problemas futuros.

Programa Padawan


O Programa Padawan é um programa de capacitação, criado pelo ecossistema Flex para devenvolver e capacitar talentos do grupo e prepara-los para futuras oportunidades internas.

Grade do Programa

O programa tem duração de 5 semanas separando por cursos em cada semana:

Calendario dos cursos:

Semana 1: Introdução a Lógica de Programção

Semana 2: Linguagem C

Semana 3: Linguagem C# e C# UT

Semana 4: Linguagem HTML

Semana 5: Linguagem Angular e Scrum

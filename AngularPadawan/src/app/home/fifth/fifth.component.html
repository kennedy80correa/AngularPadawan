Descrição do curso


No curso de Angular, tem como objetivo apresentar aos alunos o framework para construção de paginas web.

Angular é uma plataforma de aplicações web de código-fonte aberto e front-end baseado em TypeScript.

Com a conclusão desse curso os alunos devem ter a capacidade de desenvolver paginas web utilizando TypeScript.
